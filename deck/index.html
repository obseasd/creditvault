THE PROBLEM

DeFi on Creditcoin is fragmented

For Users

  • Must manually research & evaluate each yield source
  • Capital sits idle or is concentrated in a single protocol
  • No auto-compounding — missed yield from unclaimed rewards
  • Multiple transactions needed to diversify across strategies

For the Ecosystem

  • TVL is scattered across isolated protocols
  • No composable yield layer for other protocols to build on
  • New DeFi protocols struggle to attract capital
  • Lack of standardized vault interfaces (ERC-4626)

Creditcoin needs a unified yield layer that aggregates capital, optimizes returns, and provides a composable building block for the ecosystem.

ARCHITECTURE

Smart Contract Design

Contract Role Standard
CreditVault.sol Core vault — deposit, withdraw, harvest, rebalance ERC-4626
WCTC.sol Wrapped CTC — bridges native CTC to ERC-20 ERC-20
StakingStrategy.sol Staking wrapper with auto-compound IStrategy
LendingStrategy.sol Lending wrapper with auto-compound IStrategy
LPStrategy.sol LP wrapper with auto-compound IStrategy

Key Design Decisions

  • ERC-4626 standard — cvCTC shares are composable ERC-20 tokens usable as collateral across the ecosystem
  • WCTC wrapper — Bridges native CTC to ERC-20 seamlessly; users never interact with WCTC directly
  • Pluggable strategies — IStrategy interface allows swapping mock protocols for real ones without changing the vault
  • Per-block yield accrual — Mock protocols simulate realistic, time-based reward accumulation

ECOSYSTEM FIT

Why CreditVault for Creditcoin

DeFi Infrastructure

CreditVault is the first composable yield layer on Creditcoin. As new DeFi protocols launch (DEXs, lending, liquid staking), the vault can integrate them as strategies — driving TVL to the entire ecosystem from a single entry point.

ERC-4626 Composability

cvCTC shares follow the ERC-4626 standard — the same standard used by Yearn, Morpho, and EigenLayer. Any protocol on Creditcoin can accept cvCTC as collateral, building a composable DeFi stack.

TVL Growth Engine

Instead of users choosing between staking OR lending OR LP, CreditVault lets them do all three with one deposit. This increases capital efficiency and total ecosystem TVL simultaneously.

Production Ready

Pluggable IStrategy interface means zero vault changes when real protocols launch. Just deploy a new strategy adapter, call setStrategies(), and the vault seamlessly redirects capital.

CEIP Investment Fast-Track

CreditVault is purpose-built for Creditcoin's DeFi ecosystem growth. The vault architecture creates a flywheel: more strategies → better yield → more deposits → more TVL for the ecosystem.
